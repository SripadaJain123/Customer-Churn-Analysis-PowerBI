# Customer Churn Analysis Power BI Project

## Project Overview
This project focuses on Customer Churn Analysis using Power BI to identify the key factors influencing customer attrition in a telecom company.

The analysis helps understand customer behavior, churn patterns, contract risks, service dissatisfaction, and retention opportunities through interactive dashboards and business insights.

---

## Objectives
- Analyze customer churn behavior
- Identify high-risk customer segments
- Discover key churn drivers
- Improve customer retention strategies
- Build interactive Power BI dashboards
- Generate business insights for decision-making

---

## Key Insights
- Month-to-month contracts show the highest churn rate.
- Customers with Fiber Optic service are more likely to churn.
- Senior citizens have comparatively higher churn rates.
- Customers with longer tenure are more stable.
- Gender has minimal impact on churn behavior.
- Tech Support and Online Security reduce churn likelihood.

---

## Recommendations
- Improve customer onboarding during the first 6 months.
- Promote long-term contracts with discounts.
- Improve Fiber Optic service quality and pricing.
- Offer personalized support for senior citizens.
- Introduce loyalty and retention programs.
